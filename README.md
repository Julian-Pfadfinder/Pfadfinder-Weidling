# Pfadfinder Weidling Klickdummy

Statischer Klickdummy fuer die ueberarbeitete Website der Pfadfindergruppe Klosterneuburg-Weidling.

## Lokal testen

`index.html` direkt im Browser oeffnen oder den Ordner ueber einen kleinen Webserver bereitstellen.

## GitHub Pages

1. Dieses Verzeichnis in ein eigenes Git-Repository legen.
2. Auf GitHub in ein neues Repository pushen.
3. In GitHub unter `Settings > Pages` die Bereitstellung von `Deploy from a branch` aktivieren.
4. Als Branch `main` und als Ordner `/ (root)` waehlen.

Danach ist der Klickdummy unter der GitHub-Pages-URL des Repositories erreichbar.
